IVI-C Programming Guide and KtNA Driver Reference Keysight Technologies 
LabVIEW or Keysight VEE Pro

You can use the KtNA IVI-C driver with National Instruments LabVIEW or Keysight VEE Pro. To do so, you must also use Keysight Command Expert, which can generate IVI-C driver sequences that can be used directly in LabVIEW or VEE.

For additional information on using LabVIEW or VEE Pro with IVI-C drivers, see the following topics in the Command Expert help:

  • Command Expert for IVI-C
  • Command Expert for LabVIEW
  • Command Expert for VEE

Requirements

To use the KtNA IVI-C driver in LabVIEW or VEE, you must install either:

  • NI LabVIEW 2010 or newer, 32- or 64-bit
  • Keysight VEE Pro 9.0 or newer

You also need:

  • Keysight Command Expert 1.4 or newer, available for free download from the Keysight Technologies website. Install (or re-install) Command Expert after you install LabVIEW or VEE Pro.
