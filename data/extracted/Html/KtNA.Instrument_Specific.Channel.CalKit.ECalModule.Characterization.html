Attributes

Name Description
Identity Selects the model and serial number of the ECal module to be characterized. Default "Empty String".
InSituEnabled Sets or returns whether the device (CalPod module) will be characterized as an in situ device.
Number Sets and reads the number to which the User Characterization will be stored in the ECal module. Default 1.
UserDescription Sets and reads the description of the person and/or company who is producing the ECal user characterization. Default "Empty String".
VNADescription Sets and reads a description of the Vector Network Analyzer used to perform the User Characterization. Default "Empty String".
