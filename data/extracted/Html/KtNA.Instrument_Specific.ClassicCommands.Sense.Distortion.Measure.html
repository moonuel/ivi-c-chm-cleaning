Attributes

Name Description
Band Acp Lower Ibw Sets and returns the integration bandwidth (IBW) of the lower ACP measurement. Default 100MHz.
Band Acp Lower Offset Sets and returns the lower ACP measurement offset. Default -100MHz.
Band Acp Upper Ibw Sets and returns the integration bandwidth (IBW) of the upper ACP measurement. Defqault 100MHz.
Band Acp Upper Offset Sets and returns the upper ACP measurement offset. Default 100MHz.
Band Carrier Ibw Sets and returns the carrier (Signal) integration bandwidth (IBW) for the distortion measurement. Default 100MHz.
Band Carrier Offset Sets and returns the carrier offset (Signal). Offsets the Carrier integration bandwidth relative to the Carrier LO used to generate the modulation signal. Default 0.
Band Name Assigns a name to the specified distortion measurement band. Default "NewBand".
Band No Band number from 1 - 100. If unspecified, value is set to 1.
Band Notch Ibw Sets and returns the notch integration bandwidth (IBW) for an NPR Notch modulation measurement. Default 10MHz.
Band Notch Offset Sets and returns the notch offset for an NPR Notch modulation measurement. Default 0.
Band Type Set and read the modulation distortion type.
Correlation Aperture Sets and returns the frequency span window used for modeling the DUT's gain and distortion. Default 500000.
Correlation Aperture Auto Enable or disable automatic calculation of the frequency span window used for modeling the DUT's gain and distortion.
Filter Sets the measurement filter to either None (default) or RRC (root-raised-cosine filter) for EVM and ACP measurements only. Default None.
Filter Alpha Sets and returns the Alpha factor of the measurement filter.
Filter Srate Sets and returns the Symbol Rate of the filter.
Filter Srate Auto Enables or disables using the symbol rate from the file.
