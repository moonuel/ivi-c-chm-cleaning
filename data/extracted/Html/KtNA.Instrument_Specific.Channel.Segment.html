Attributes

Name Description
Arbitrary Enabled Enables you to setup a segment sweep with arbitrary frequencies.
Bandwidth Control Enabled Specifies whether the IF Bandwidth resolution can be set independently for each segment.
Enabled Turns the specified sweep segment ON or OFF. At least ONE segment must be ON or Sweep Mode is automatically set to Linear.
IF Bandwidth Sets the IF Bandwidth for the specified segment.
Power Control Enabled Specifies whether Power Level can be set independently for each segment.
Segment No Segment number to add, delete or modify.
Sweep Dwell Time Sets or returns the sweep dwell time of the specified sweep segment. Default 0s.
Sweep Dwell Time Control Enabled Turns ON or OFF the sweep dwell time of the specified sweep segment.
X Spacing Sets X-axis spacing ON or OFF. Default Linear.
