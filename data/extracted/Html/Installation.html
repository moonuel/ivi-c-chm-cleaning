IVI Driver Installation

Updating to a New Driver Version

Only a single version of an IVI-C driver can be installed on a system at once. This rule is dictated by the IVI specifications. To ensure that only a single version of the KtNA driver exists on the system, the KtNA installer automatically uninstalls any previous version of the driver. Although you can always explicitly uninstall the driver, this is typically not necessary.

File Locations

The KtNA installer deploys a variety of files on the target computer. The notation <IviInstallDir> is used to denote the directory that contains the IVI Shared Component (default is C:\<Program Files>\IVI Foundation\IVI). Substitute the install directory if your IVI Shared Components are in an alternate location.

File Description
<IviInstallDir>\Bin\KtNA.dll Driver DLL.
<IviInstallDir>\Include\KtNA.h Driver header file.
<IviInstallDir>\Lib\msc\KtNA.lib Microsoft-format import library.
<IviInstallDir>\Drivers\KtNA\KtNA.fp Function panel file.
<IviInstallDir>\Drivers\KtNA\KtNA.sub Attribute information file.
<IviInstallDir>\Drivers\KtNA\KtNA.chm HTML 1.0 format help file.
<IviInstallDir>\Drivers\KtNA\KtNA.msha Help Viewer manifest file.
<IviInstallDir>\Drivers\KtNA\KtNA.cab Help Viewer signed content cabinet file.
<IviInstallDir>\Drivers\KtNA\Examples\ Driver example programs.
<IviInstallDir>\Drivers\KtNA\Source\ Optionally installed driver source code project.

IVI Shared Components

Before any IVI driver may be installed on a system, the IVI shared components must be properly installed. Keysight IO Libraries Suite (which must be installed before you install your Keysight driver) installs these shared components.

For additional information on the IVI shared components, visit the IVI Foundation web site.

To determine which version of the shared components are installed on your system:

  1. Open the <IviInstallDir>\Bin folder (typically C:\Program Files\IVI Foundation\IVI\Bin).
  2. Right-click IviSharedComponentVersion.dll and choose Properties.
  3. In the Properties dialog, click the Version tab.

The File version field indicates which version of the IVI shared components are installed.

Uninstallation

The only proper way to explicitly remove the driver from the system is to run  the uninstallation program. Simply removing the installed files does not properly uninstall the driver.

To uninstall the driver:

  1. From the Windows Start menu, open the Control Panel.
  2. Double-click Add or Remove Programs.
  3. Select the driver and click Remove.
Note:

In order to prevent loss of important user configuration information between uninstall/reinstall operations, the uninstallation program does not remove any of the following items from the IVI Configuration Store:

  • Logical Name
  • Driver Session
  • Hardware Asset

The Software Module is the only IVI Configuration Store item that is deleted during an uninstall.

See IVI Configuration Store for more information on Config Store entries.
