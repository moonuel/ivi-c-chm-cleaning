Attributes

Name Description
Bucket Sets and reads the data point (bucket) number of the trace on which the marker resides.
Coupling Method Sets and reads the scope of Coupled Markers.
Coupling State Sets and reads the state of Coupled Markers (ON and OFF).
Delta Specifies whether marker is relative to the Reference marker or absolute.
Discrete Makes the specified marker display either a calculated value between data points (interpolated data) or the actual data points (discrete data).
Distance Set or query marker distance on a time domain trace.
Format
Marker No Any marker number from 1 to 15. Default 1.
Reference State Turns ON or OFF the reference marker mode, for the active trace of selected channel.
Reference X Sets and returns the absolute x-axis value of the reference marker.
State Turns ON or OFF the specified marker. Marker 16 is the Reference Marker.
X Set or get the marker's X-axis value (frequency, power, or time).
