Attributes

Name Description
Center Sets or returns the power sweep center power.
Coupling Enabled Turns Port Power Coupling ON or OFF.
Enabled Turns RF power from the source ON or OFF.
Level Sets or returns the RF power output level in dBm.
Mode Sets the state of VNA source for the specified port.
Slope Sets the RF power slope value in in db/GHz.
Slope Enabled Turns Power Slope ON or OFF.
Span Sets the power sweep span power in dBm.
Start Sets or returns the power sweep start power for ALL ports being used by the specified channel.
Start For Specific Port Sets and reads the power sweep start power value in dBm for a specific port. This allows uncoupled forward and reverse power sweep ranges.
Stop Sets or returns the power sweep stop power for ALL ports being used by the specified channel.
Stop For Specific Port Sets and reads the power sweep stop power value in dBm for a specific port. This allows uncoupled forward and reverse power sweep ranges.
