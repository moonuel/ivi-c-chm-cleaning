Attributes

Name Description
Attenuation Sets the attenuation level for the selected channel. The range is from 0 to 30 dB with the step size 1 dB. Default 10.
AttenuationAutoEnabled Enable and disable the automatic attenuation control. Default 1 (On).
BasebandGain Sets and returns the setting for baseband gain. Default 0.
BasebandInput Sets and returns the setting for baseband input. Default ACLF.
CarrierFrequency Sets and returns the carrier frequency. Default 1 GHz.
CarrierSearchEnabled Enables and disables a broadband carrier search within the range specified. Default 1 (On).
CarrierSearchThreshold Sets and returns the threshold to use during a carrier search. Default -20dBm.
CompositionRange Choose the range of AM/PM composite measurement. Default R30.
CrossCorrelationCount Sets and returns the cross correlation count number. Correlation count number from 1 to 100,000. Default 1.
DecompositionModel Choose decomposition model of AM/PM composite measurement range. Default 4.
FFTAverageFactor Sets and returns the FFT average factor number. FFT average factor number from 1 to 10,000. Default 1.
Noise Sets and returns the noise type to phase, residual or baseband noise. Default PhaseNoise.
OffsetFrequencyExtension Sets and returns the Offset frequency extension status. Default 0 (Off).
PowerCorrectionEnabled Enable and disable the power correction. Default 0 (Off).
PulseEnabled Enables or disables Pulse setup for phase noise measurement. Default 0 (Off).
PulsePeriod Set or returns Pulse period for phase noise measurement. Default 100E-6.
ResolutionBWRatio Sets and returns the resolution bandwidth ratio, which is the specified resolution bandwidth percentage of every half decade offset frequency. Default 10.
SweepNoiseMode Sets and returns the sweep noise mode. Default Normal.
SweepType Sets and returns the phase noise sweep type to logarithmic or segment.
XCorrelationGainIndicatorEnabled This command turns the cross-correlation gain indicator on and off. Default 0 (Off).
