Attributes

Name Description
CW Time Auto Enabled Sets and returns the state of automatic CW sweep time (used in Pulse Profile mode).
Delay Auto Enabled In Narrowband pulse mode, choose to set the delay and width automatically or manually.
Drive Mod Source Auto Enabled In Narrowband pulse mode, set the drive for the source modulation automatically or manually.
If Gain Auto Enabled In Narrowband pulse mode, set the IF Gain automatically or manually.
Mode Sets and returns the pulse measurement state for the channel.
Mode Auto Enabled Set pulse mode automatically or manually (Narrowband or Wideband) for the channel.
PRF Auto Enabled In Narrowband pulse mode, choose to set the Pulse Repetition Frequency automatically or manually.
Primary Clock Sets and returns the Primary Clock setting and is controlled by the internal or external pulse generator which is the primary pulse clock.
Primary Frequency Sets and returns the primary pulse measurement frequency. Default 1KHz.
Primary Period Sets and returns the primary pulse period in seconds. Default 1 msec.
Primary Width Sets and returns the primary pulse width in seconds. Default 100 microseconds.
Shape Sets and returns the pulse shape type.
Software Gating Enabled When set to OFF, the improved software gating sensitivity is turned OFF and all data outside the measurement band is zeroed.
Start Time Sets and returns the start time of the pulse in seconds. Default 0.
Stop Time Sets and returns the stop time of the pulse in seconds.
Wideband Mode Enabled Set and read the pulse mode detection method.
