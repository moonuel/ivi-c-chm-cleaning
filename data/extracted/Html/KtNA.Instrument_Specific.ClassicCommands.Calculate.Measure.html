Attributes

Name Description
ComputationDeviation Calculates the deviation from a least-squares best fit line.
ConversionFunction Sets or gets the parameter after conversion using the parameter conversion function, for the active trace of selected channel. Default Off.
Format Sets the display format for the measurement. Default Log.
FrequencyReference Set and read the frequency reference for delta frequency in VCO Characteristic measurement. Default 0.
FrequencySensitivityAperture Set and read the sensitivity aperture for Freq/V in VCO Characteristic measurement. Default 1.
HoldType Sets the type of trace hold to perform.
Measurement No Measurement number for each measurement. Choose from 1 to 2000. Default 1.
Mixer X Axis Sets or returns the swept parameter to display on the X-axis for the selected FCA and GCX measurement. Default Input.
OffsetMagnitude Offsets the data trace magnitude by the specified value. Default 0.
OffsetMagnitudeSlope Offsets the data trace magnitude to a value that changes linearly with frequency. The offset slope begins at 0 Hz.
OffsetPhase Sets the phase offset for the selected measurement. Default 0 degrees.
