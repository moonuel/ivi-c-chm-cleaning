Attributes

Name Description
Average Count Set and read the averaging factor for the noise receiver.
Average State Turns noise averaging ON and OFF.
Bwidth Set and read the bandwidth of the noise receiver.
Gain Set and read the amount of gain for the noise receiver.
Gain Ctcheck Turns noise threshold checking ON and OFF.
Impedance Count Sets the number of impedance states to use during calibrated measurements.
Narrowband State Turns narrowband noise figure compensation ON and OFF.
Pull State Enables and disables the use of source pull technique to compute S22.
Receiver
Temperature Ambient Sets the temperature at which the current noise measurement is occurring. Default 295 Kelvin.
Temperature Ambient Auto If ON, will use 302K as the ambient temperature when vector correction is applied and the tuner is an ECal or internal tuner. If OFF, will use the specified ambient temperature.
