Attributes

Name Description
Bandwidth Set and read the resolution bandwidth.
Bandwidth Resolution Auto Set and read how the resolution bandwidth is set. When ON, the resolution bandwidth is set based on Span or RBW ratio.
Bandwidth Search Occupied Min Set and read the minimum search frequency to use during an Occupied BW search measurement. Default 250MHz.
Bandwidth Shape Set and read the resolution bandwidth shape. Default Gaussian.
Bandwidth Video Set and read the video bandwidth.
Bandwidth Video Auto Set and read how the video bandwidth is set. When ON, video bandwidth is set based on RBW or VBW ratio.
Bandwidth Video Aver Type Set and read the averaging type. Default Power.
Bandwidth Video Ratio Set and read the RBW or VBW ratio. Default 1.0
Detector Bypass Set and read the ON or OFF state of the detector bypass setting.
Detector Function Set and read the detector type. Default Peak.
Dft Bandwidth Auto Set and read the default values for DFT bandwidth.
Dft Bandwidth Narrow Max Set and read the maximum value for narrow DFT bandwidth.
Dft Bandwidth Narrow Min Set and read the minimum value for narrow DFT bandwidth.
Dft Bandwidth Wide Max Set and read the maximum value for wide DFT bandwidth.
Dft Bandwidth Wide Min Set and read the minimum value for wide DFT bandwidth.
Dft Type Set and return the DFT record size type. Default Radix.
Fft Dither Set and read the ON or OFF state of the FFT grid dither setting
Frequency Converter Set and read the ON on OFF state of the dual band configuration for fast setting of the frequency converter DUT.
Frequency Converter Input Center Set and read the frequency converter DUT input center frequency in Hz.
Frequency Converter Input Port Set and read the port number of the VNA connected to the input of the frequency converter DUT. Default 0.
Frequency Converter Input Source Set and read the source name that generates the modulated signal. Default "None".
Frequency Converter Output Center Set and read the frequency converter DUT output center frequency in Hz.
Frequency Converter Output Inverted Enable or disable inverting the spectrum (or mirror) by the frequency converter operation.
Frequency Converter Output Offset Set and read the LO offset value in Hz. Default 0.
Frequency Converter Output Port Set and read the port number of the VNA connected to the output of the frequency converter DUT. Default 0.
Frequency Span Bandwidth Ratio Set and read the Frequency Span or RBW ratio.
Image Reject Set and read the image reject mode. Default Normal.
Image Strength Sets and returns the image rejection strength. Default Medium
Lo Random Set and read the LO randomize state.
Lobaseband State Enables and disables baseband sweep independent of the LO sweep to allow signals down to 1 Hz to be analyzed.
Loforce Set and read enable force LO to frequency mode.
Loforce Frequency Set and read enable force LO to the specified frequency in Hz.
Loforce Offset Divider Set and read the divider in the equation for setting LO frequency. Default 1.
Loforce Offset Multiplier Set and read the multiplier in the equation for setting LO frequency. Default1.
Loforce Offset Source Set and read the source name in the equation for setting LO frequency.
Trace Image Set and read the show or hide state of the image reject traces in the measurement parameters dialog.
Trigger Level Set and read the ON or OFF state of a measurement trigger event that will occur whenever the ADC level is greater than the value specified
Trigger Level Value Set and read the ADC trigger level.
Trigger Percounter Set and read the ON or OFF state of a measurement trigger event based on the specified period.
Trigger Percounter Value Set and read the periodic counter value. This command initiates a measurement trigger event based on the specified period.
