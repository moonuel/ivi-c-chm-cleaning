Functions

Name Description
Get DC Receiver Catalog Returns a list of names of the internal DC receivers.
Get DC Source Catalog Returns a list of names of the internal DC sources.
Get Internal Port Catalog Returns a list of internal test port names.
Get Internal Port Count Returns the number of internal test ports.
Get Internal Source Port Catalog Returns a list of internal source port names.
Get Internal Source Port Count Returns the number of internal source ports.
Get Max Discrete Power Returns a single max leveled power value (in dBm) indicating the most restrictive maximum for all discrete maximum powers (the minimum of all max leveled powers).
Get Max Power Range Returns the minimum of all max leveled power values (in dBm) from range start frequency to stop frequency (inclusive).
Get Max Receiver Attenuation Returns the maximum amount of receiver attenuation on the specified port.
Get Max Source Attenuation Returns the maximum amount of source attenuation on the specified port.
Get Min Discrete Power Returns a single minimum power value (in dBm) indicating the most restrictive minimum for all discrete minimum powers (the maximum of all minimum powers).
Get Min Power Range The maximum of all min power values (in dBm) from range start frequency to stop frequency (inclusive).
Get No OfDC Receiver Returns the number of internal DC receivers in the analyzer.
Get No OfDC Source Returns the number of internal DC sources in the analyzer.
Get Port Catalog Returns a list of test port names including external testset ports.
Get Port Count Returns the number of test ports including external testset ports.
Get Port No Returns the port number associated with the specified port name.
Get Receiver Attenuator Step Size Returns the step size of the receiver attenuator on the specified port.
Get Receiver Count Returns the number of receivers in the analyzer.
Get RF Path Element Returns the name of the value for the given path element name.
Get RF Path Element Catalog Returns a string with the names of all valid RF path configuration elements.
Get RF Path Element Value Catalog Returns all valid values for the given path configuration element.
Get Source Attenuator Step Size Returns the step size of the source attenuator on the specified port.
Get Source Count Returns the number of source ports, including any configured external sources.
Get Source Port Catalog Returns a list of source port names, including any configured external sources.
Get Source Port Count Returns the number of source ports, including any configured external sources.
Get System Capabilities Hardware Power Range Port Gets the value of the SYSTEM_CAPABILITIES_HARDWARE_POWER_RANGE_PORT attribute.
Get System Capabilities Hardware Power Range Start Frequency Gets the value of the SYSTEM_CAPABILITIES_HARDWARE_POWER_RANGE_START_FREQUENCY attribute.
Get System Capabilities Hardware Power Range Stop Frequency Gets the value of the SYSTEM_CAPABILITIES_HARDWARE_POWER_RANGE_STOP_FREQUENCY attribute.
Get System Capabilities Hardware Power Range Type Gets the value of the SYSTEM_CAPABILITIES_HARDWARE_POWER_RANGE_TYPE attribute.
Is Direct Receiver Access Present Returns whether or not the analyzer has direct receiver access (front-panel jumpers).
Is Gain Coupling Controllable Returns whether or not the gain coupling can be changed.
Is LF Extension Present Returns whether or not the VNA has the low frequency extension (LFE) installed.
Is Receiver Attenuator Present Returns whether or not there is a receiver attenuator on the specified port.
Is Reference Bypass Switch Present Returns whether or not the specified port number has a reference bypass switch.
Query Discrete Frequencies Returns the list of discrete frequencies corresponding to the powers returned by the discrete min and max power list functions.
Query Max Discrete Power List Returns an array of max leveled power values (in dBm), where each element corresponds to the maximum leveled power possible for CW stimulus at the corresponding frequency
Query Min Discrete Power List Returns an array of minimum power values (in dBm), where each element corresponds to the minimum power possible for CW stimulus at the corresponding frequency set
Reset Power Resets all power range properties to default values, as if the instrument had been preset.
Set Discrete Frequencies Sets the list of discrete frequencies corresponding to the powers returned by the discrete min and max power list functions.
Set RF Path Element Sets the value of a path element.
Set System Capabilities Hardware Power Range Port Sets the value of the SYSTEM_CAPABILITIES_HARDWARE_POWER_RANGE_PORT attribute.
Set System Capabilities Hardware Power Range Start Frequency Sets the value of the SYSTEM_CAPABILITIES_HARDWARE_POWER_RANGE_START_FREQUENCY attribute.
Set System Capabilities Hardware Power Range Stop Frequency Sets the value of the SYSTEM_CAPABILITIES_HARDWARE_POWER_RANGE_STOP_FREQUENCY attribute.
Set System Capabilities Hardware Power Range Type Sets the value of the SYSTEM_CAPABILITIES_HARDWARE_POWER_RANGE_TYPE attribute.
