Attributes

Name Description
Couple Divisor Sets and reads the value by which the coupled range will be divided to achieve the frequency range specified.
Couple Multiplier Sets and reads the value by which the coupled range will be multiplied to achieve the frequency range specified
Couple Offset Sets and reads the frequency range number to be used as an offset.
CoupleID Sets and reads the frequency range to couple settings to. Default 1.
Frequency Range Coupling Enabled Sets and reads the ON or OFF state of frequency range coupling.
Frequency Range No Frequency range number. It starts from 1.
Frequency Range Start Sets and reads the start value for the specified frequency range.
Frequency Range Stop Sets and reads the stop value for the specified frequency range.
Frequency RangeIFBW Sets and reads the receiver IF bandwidth setting in Hz. Default 1e5.
Up Conversion Enabled Sets and reads the state of the Up or Down conversion setting.
