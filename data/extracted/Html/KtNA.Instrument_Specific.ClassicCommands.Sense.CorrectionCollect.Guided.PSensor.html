Attributes

Name Description
Ckit Specifies the calibration kit to be used when an adapter is necessary to connect the power sensor to the port during a guided calibration.
Connector Specifies the connector type for a power sensor when a power calibration is part of the Guided Calibration process.
Item Number Power sensor item number.
Multiple Ckit Set and read the Cal Kit to be used when an adapter is necessary to connect the power sensor to the cal plane and you choose to remove its effects from the measurement.
Multiple Connector Set and read the connector type of a power sensor to be used when an adapter is necessary to connect the power sensor to the cal plane and you choose to remove its effects from the measurement.
Multiple Frequency Start Set and read the start frequency for the specified power sensor (port number).
Multiple Frequency Stop Set and read the stop frequency for the specified power sensor (port number).
Multiple Name Set and read the name of the power sensor.
Multiple State Enables the use of multiple power sensors to calibrate the entire frequency span of the channel.
Port No Source port number of the VNA. Choose any number between 1 and 50. Default 1.
Power Level Specifies the power level in dBm at which to perform a power calibration during a guided calibration. Default 0.
State Enables Guided Power Cal and sets the source port to be calibrated.
