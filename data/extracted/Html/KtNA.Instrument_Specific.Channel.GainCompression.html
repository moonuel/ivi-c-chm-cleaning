Attributes

Name Description
Acquisition Mode Set and read the method by which gain compression data is acquired. Default SmartSweep
Algo Method Set and read the algorithm method used to compute gain compression. Default LinearGain.
Backoff Set and read value for the BackOff compression algorithm in dB.
Coarse Power Adjustment Set and read the Safe Sweep COARSE power adjustment in dBm.
CompressionDC Reading Enabled Set and read the DC readings at the compression point in the last iteration of a smart sweep.
DC Device Max Limit Set and read the maximum limit of the external DC device. Default -5.
DC Device Name Set and read the name of the external DC device.
DeltaX Set and read the 'X' value in the delta X or Y compression algorithm.
DeltaY Set and read the 'Y' value in the delta X or Y compression algorithm.
End Of Sweep This setting is used to protect a sensitive device from too much power during the sweep retrace. Default Standard.
Fine Power Adjustment Set and read the Safe Sweep FINE power adjustment in dBm.
Interpolation Enabled Sets whether or not interpolation should be performed on 2D measured compression data.
Level Set and read the desired gain reduction (from reference gain).
Linear Gain Input Power Set and read the input power at which Linear Gain and all S-parameters are measured.
Linear Input Power Aperture Set and read the aperture to use when computing the linear input power.
Max Power Limit When the VNA port that is connected to the DUT Output measures this value, the input power to the DUT is no longer incremented at that frequency.
No Of Frequency Points Set and read the number of data points in each frequency sweep.
No Of Power Points Set and read the number of data points in each power sweep. Default 21.
Phase Level Set and read the desired phase to measure compression.
Phase Mode Set and read compression format to be either magnitude, phase, or magnitude and phase. Default Magnitude.
Power Smoothing Aperture Set and read the power smoothing aperture in percent. Default 25.
Power Smoothing Enabled Set and read the state of the power smoothing.
Reverse Power Set and read the reverse power level to the DUT.
Safe Sweep Enabled Set and read the (ON or OFF) state of Safe Sweep mode.
Safe Threshold Set and read the compression level in which Safe Sweep changes from the COARSE power adjustment to the FINE power adjustment.
Same Path Config Enabled This function is used when GCX channels with a reference mixer and one GCA channel without a reference mixer are set up simultaneously and you want to use the same path configuration for all channels. This remaps the source port to Port 1.
Saturation Level Set and read the deviation dB from the maximum Pout. This is the point of saturation.
Smart Sweep Max No Of Iteration Set and read the maximum permitted number of iterations which SMART Sweep may utilize to find the desired compression level, to within the specified tolerance.
Smart Sweep Results Enabled Set and read enable for showing intermediate results for each iteration of SMART Sweep
Smart Sweep Settling Time Set and read the amount of time SMART Sweep will dwell at the first point where the input power changes by the Backoff or X level.
Smart Sweep Tolerance Set and read the acceptable range SMART Sweep will allow for the measured compression level.
Start Power Set and read the start power level.
Stop Power Set and read the stop power level.
