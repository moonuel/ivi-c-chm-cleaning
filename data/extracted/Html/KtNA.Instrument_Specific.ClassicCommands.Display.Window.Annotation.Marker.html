Attributes

Name Description
Couple State Sets the marker readouts to coupled (one combination annotation) or not coupled (one annotation per trace). This setting is per Window scope.
Measure X Position Sets or returns the X-axis position of marker readouts. Readouts are right-justified at the specified position.
Measure Y Position Sets or returns the Y-axis position of marker readouts. Readouts are top-justified at the specified position.
Number Sets the number of marker readouts to display per trace. Display up to 20 marker readouts per window.
Resolution Response For the Y-axis (response), sets the number digits to display after the decimal point in marker readouts.
Resolution Stimulus For the X-axis (stimulus), sets the number digits to display after the decimal point in marker readouts.
Size Specifies the size of the marker readout text.
State Specifies whether to show or hide the Marker readout (when markers are ON) on the selected window.
Symbol Sets the symbol to display for marker position.
Symbol Above Specifies whether or not to force marker symbols to be displayed above the trace.
Visible Shows the marker readouts only for active trace or for all traces. This setting is per Window scope.
X Position Sets the X-axis position of marker readouts. Readouts are right-justified at the specified position.
Y Position Sets the Y-axis position of marker readouts. Readouts are top-justified at the specified position.
