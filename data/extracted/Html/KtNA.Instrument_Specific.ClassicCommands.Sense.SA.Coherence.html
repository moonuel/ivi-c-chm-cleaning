Attributes

Name Description
Lofthru Monitor Set and read the LO feedthru monitoring state.
Lofthru Monitor Receiver Set and read the target receiver for LO Feedthru monitor. Select the reference receiver in most cases. Default "a1".
Lofthru Monitor Tolerance Set and read the threshold level for warning. Default -40.
Lofthru Monitor Type Set and read the monitoring method for LO feedthru monitor. Default Band power.
Multitone Data Set and read the data display mode. Default All.
Multitone Hreject Set and read the number of test signal harmonics you want to be protected against. This adds constraints to the list of LOs used to cover the span. Default 0.
Multitone Nyqreject Sets and returns the Nyquist protection level. Default 0.
Multitone Period Set and read the test signal repetition rate (in seconds). Default 1E6.
Multitone Reference Set and read the multitone image rejection offset frequency. Default 0.
Multitone Spacing Set and read the tone spacing of the multitone signal (in Hz). Default 1000000.
Multitone State Enables or disables multitone image rejection.
Phase Display Level Set and read the phase display minimum level in dBm. Default -60 dBm.
Phase State Enables or disables phase computing.
Phase Stitching Enable or disable phase stitching.
Phase Stitching Common Enables or disables selected receiver, or receiver with highest energy in the overlap area, to drive the phase stitching for all receivers.
Phase Stitching Hw Tstamp Enables or disables phase stitching using hardware time stamps to determine the phase of each LO acquisition and stitch them back together with phase consistency.
Phase Stitching Level Set and read the minimum tone power level to perform phase stitching in dBm. Default -150 dBm.
Phase Stitching Receiver Auto Set and read the receiver for stitching automatically.
Phase Stitching Receiver Name Assigns a specified receiver for phase stitching. Default R1.
Pulse Search Count Set and read the maximum pulse search count. Default 100.
Pulse Search Duty Tolerance Set and read the duty cycle tolerance.
Pulse Search Period Set and read the period starting point in seconds.
Pulse Search Period Tolerance Set and read the period tolerance. Only solutions that ensure this tolerance will be accepted.
Pulse Search Priority Set and read the priority choice for the search algorithm. Default Width.
Pulse Search Width Set and read the width starting point in seconds.
Pulse Search Width Tolerance Set and read the width tolerance. Only solutions that ensure this tolerance will be accepted.
Vector Average Set and read the ON or OFF state of the vector averaging.
Vector Average Value Set and read the vector averaging value.
