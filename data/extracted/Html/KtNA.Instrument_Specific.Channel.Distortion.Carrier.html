Attributes

Name Description
NBW Mode Selects the noise bandwidth mode in the power sweep list table. Default Fixed.
No Of Points Sets and returns the number of power points to measure for the power sweep list measurement. Default 11.
NoiseBW Sets and returns the noise bandwidth setting used for the specified row in the power sweep list table. Default 100Hz.
NoiseBW Auto Enabled Enable or disable increasing noise bandwidth at high powers automatically for faster measurements.
Power Level Sets and returns the power level used for the specified row in the power sweep list table. Default -10 dBm.
Ramp Points Sets and returns the number of power points to measure for the power sweep ramp measurement. Default 11.
Ramp Start Sets and returns the starting power level for the power sweep ramp measurement. Default -20dBm.
Ramp Stop Sets and returns the stop power level for the power sweep ramp measurement. Default -10dBm.
Receiver Attenuation Sets and returns the receiver attenuation for the specified row in the power sweep list table. 0dB.
Receiver Attenuation Mode Selects the receiver attenuation mode in the power sweep list table. Default Fixed.
Source Attenuation Sets and returns the source attenuation for the specified row in the power sweep list table. Default 0dB.
Source Attenuation Mode Selects the source attenuation mode in the power sweep list table. Default Fixed.
Sweep No Power sweep row number. If unspecified, value is set to 1.
Sweep Type Selects between a ramp type power sweep or a list of power values to define a power sweep. Default Ramp.
