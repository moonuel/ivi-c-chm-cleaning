Attributes

Name Description
Band Name Assigns a name to the specified distortion measurement band. Default "NewBand".
Band No Band number from 1 - 100. If unspecified, value is set to 1.
Band Type Set and read the modulation distortion type.
Carrier Offset Sets and returns the carrier offset (Signal). Offsets the Carrier integration bandwidth relative to the Carrier LO used to generate the modulation signal. Default 0.
CarrierIBW Sets and returns the carrier (Signal) integration bandwidth (IBW) for the distortion measurement. Default 100MHz.
Correlation Aperture Sets and returns the frequency span window used for modeling the DUT's gain and distortion. Default 500000.
Correlation Aperture Auto Enabled Enable or disable automatic calculation of the frequency span window used for modeling the DUT's gain and distortion.
Filter Alpha Factor Sets and returns the Alpha factor of the measurement filter.
Filter Symbol Rate Sets and returns the Symbol Rate of the filter.
Filter Type Sets the measurement filter to either None (default) or RRC (root-raised-cosine filter) for EVM and ACP measurements only. Default None.
LowerACP Offset Sets and returns the lower ACP measurement offset. Default -100MHz.
LowerACPIBW Sets and returns the integration bandwidth (IBW) of the lower ACP measurement. Default 100MHz.
Notch Offset Sets and returns the notch offset for an NPR Notch modulation measurement. Default 0.
NotchIBW Sets and returns the notch integration bandwidth (IBW) for an NPR Notch modulation measurement. Default 10MHz.
Symbol Rate Auto Enabled Enables or disables using the symbol rate from the file.
UpperACP Offset Sets and returns the upper ACP measurement offset. Default 100MHz.
UpperACPIBW Sets and returns the integration bandwidth (IBW) of the upper ACP measurement. Defqault 100MHz.
