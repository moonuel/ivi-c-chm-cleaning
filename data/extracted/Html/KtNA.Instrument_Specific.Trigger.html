Attributes

Name Description
Connector For External Input Specifies the connector to use for the external trigger input.
Connector For Output Ready Line Specifies the connector to use for the trigger OUT ready line.
Delay Sets and reads the trigger delay in seconds for ALL channels (globally).
Initiate Continuous Enabled Specifies whether the VNA trigger source is set to Internal (continuous) or Manual.
Method Specifies the type of EXTERNAL trigger input detection used to listen for signals on the Meas Trig IN connectors. Edge triggers are most commonly used.
Ready Manual Enabled Sets the mode of Ready for Trigger output.
Scope Specifies whether a trigger signal is sent to all channels or only the current channel.
Slope Specifies the polarity expected by the external trigger input circuitry.
Source Sets the source of the sweep trigger signal.
