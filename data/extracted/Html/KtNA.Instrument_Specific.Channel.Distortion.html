Attributes

Name Description
ADC Filter Type Selects between a narrow and wide IF filter anti-aliasing path.
Amplifier Gain Sets and returns the nominal gain of an external source amplifier. Default 0.
Attenuator Included Enables or disables the external modulated source to pass through the attenuator of the specified PNA source and the attenuator setting is used to calculate the power levels.
Carrier Frequency Sets and returns the carrier center frequency. Default 1.5GHz.
Carrier Power Level For Test Sets and returns the carrier power level used for the distortion test at either the input or output of the DUT. Default -10dBm.
DUT Gain Sets and returns the nominal DUT gain. Default 0.
DUT Input Port No Sets and returns the VNA port number connected to the DUT input. Default 1.
DUT Mode Set and read where to apply power to the DUT (input or output). Default DIN1.
DUT Noise Figure Sets and returns the nominal DUT noise figure. This value is used by the EVM measurement. Default 0.
DUT Output Port No Sets and returns the VNA port number connected to the DUT output. Default 2.
EVM Scaling Factor Sets and returns the scaling factor applied to the EVM measurements.
Mod Source Name Sets and returns the distortion modulation source name.
Noise Figure Included Include or exclude the nominal DUT noise figure value. Default 0.
Receiver Timing Auto Enabled Enables or disables receiver timing auto generation from the source pulse timing.
S ParameterBW Sets and returns the IF bandwidth for the linear S21 sweep. Default 1 kHz
Sparameter Enabled Enable or disable S-parameter sweep. If disabled, curve fitting (smoothing) is used instead of the S-parameter sweep.
Sparameter Power Level Sets and returns the power level used to measure the DUT S-parameters when operating in its linear region. Default -30dBm.
Sparameter Reuse Enabled Enable or disable the re-use of the previous linear S-parameter measurements if available and skip sweep.
Sparameter Step Size Sets and returns the step size for the linear S21 sweep. Default 1 MHz.
Sweep Delay Sets and returns the delay in seconds before the test signal to allow the RF source to settle. Default 0s.
Sweep Type Selects between a fixed or power sweep type of sweep. Default Fixed.
