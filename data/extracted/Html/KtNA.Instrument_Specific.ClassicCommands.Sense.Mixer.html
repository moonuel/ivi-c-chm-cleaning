Attributes

Name Description
Elo State Sets and returns the ON or OFF state of Embedded LO.
If Frequency Sideband When two LO stages are used, sets or returns whether to select the sum or difference for the IF1 product.
If Frequency Start Sets or returns the IF start frequency value of the mixer.
If Frequency Stop Sets or returns the stop frequency value of the mixer IF frequency.
Input Frequency Denominator Sets or returns the denominator value of the Input Fractional Multiplier.
Input Frequency Fixed Sets or returns the fixed frequency of the input.
Input Frequency Mode Sets or returns the Input sweep mode. Default Fixed.
Input Frequency Numerator Sets or returns the numerator value of the Input Fractional Multiplier.
Input Frequency Start Sets or returns the Input start frequency value of the mixer.
Input Frequency Stop Sets or returns the Input stop frequency value of the mixer.
Input Power Sets or returns the value of the Input Power.
Input Power Start Sets the input start power for a power sweep in a mixer channel like SMC. The value is only used when the sweep type is power sweep.
Input Power Stop Sets the input stop power for a power sweep in a mixer channel . The value is only used when the sweep type is power sweep.
Input Power Use Nominal Toggles the Use Nominal Incident Power setting ON and OFF. This setting is ONLY to be used with SMC measurements.
Normalize Point Sets or returns the data point for normalizing the phase measurement.
Output Frequency Fixed Sets or returns the output fixed frequency of the mixer.
Output Frequency Mode Sets or returns the Output sweep mode. Default Fixed.
Output Frequency Sideband Specify whether to select the sum (High) or difference (Low) products.
Output Frequency Start Sets or returns the Output start frequency of the mixer.
Output Frequency Stop Sets or returns the Output stop frequency of the mixer.
Phase Absolute State Sets and returns the state of SMC Phase measurements to use absolute phase
Phase State Sets and returns the state of SMC Phase measurements and calibrations.
Stage Number of IF stages (LOs) used in the mixer.
X Axis Sets or returns the swept frequency range to display on the X-axis for the IMDx or NFx channel. Default Output.
