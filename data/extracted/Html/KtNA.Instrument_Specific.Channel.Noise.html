Attributes

Name Description
Ambient Temperature Sets the temperature at which the current noise measurement is occurring. Default 295 Kelvin.
Ambient Temperature Auto Enabled If ON, will use 302K as the ambient temperature when vector correction is applied and the tuner is an ECal or internal tuner. If OFF, will use the specified ambient temperature.
Averaging Enabled Turns noise averaging ON and OFF.
Averaging Factor Set and read the averaging factor for the noise receiver.
Bandwidth Set and read the bandwidth of the noise receiver.
Gain Set and read the amount of gain for the noise receiver.
Impedance State Count Sets the number of impedance states to use during calibrated measurements.
Narrowband Enabled Turns narrowband noise figure compensation ON and OFF.
Receiver Sets and returns the noise receiver to use for noise measurements. Default Noise.
Source Pull Enabled Enables and disables the use of source pull technique to compute S22.
Threshold Checking Enabled Turns noise threshold checking ON and OFF.
