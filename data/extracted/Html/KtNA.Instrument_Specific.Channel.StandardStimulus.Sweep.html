Attributes

Name Description
Coupling Mode Sets the sweep mode as Chopped or Alternate.
CW Frequency Sets or returns the Continuous Wave (or Fixed) frequency.
Delay Sets or returns the time in seconds to wait just before acquisition begins for each sweep.
Dwell Time Sets or returns the dwell time in seconds between each sweep point.
Dwell Time Auto Enabled Specifies whether or not to automatically calculate and set the minimum possible dwell time.
Fast Mode Sets and returns the state of Fast Sweep mode.
FastCWPoints Enables Fast CW sweep and sets the number of data points for the channel.
LF Extension Enabled Turns ON or OFF low frequency extension for extending the range down to 1 kHz start frequency.
Mode Sets sweep as Stepped or Analog.
Number Of Points Sets or returns the number of data points for the measurement.
Point Sweep Mode Enabled Turns ON and OFF point sweep mode.
Shift LO Max Frequency Sets the Shift LO maximum frequency for the selected channel.
Shift LO Mode Enabled Turns ON or OFF the Shift LO mode for the selected channel.
Time Sets or Returns the time in seconds the analyzer takes to complete one sweep.
Time Auto Enabled Turns the automatic sweep time function ON or OFF.
Trigger Count Sets the trigger count (groups) for the specified channel. Set trigger mode to group after setting this count.
Trigger Mode Sets the number of trigger signals the specified channel will ACCEPT.
