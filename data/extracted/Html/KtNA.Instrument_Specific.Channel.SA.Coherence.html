Attributes

Name Description
ADC Trigger Enabled Set and read the ON or OFF state of a measurement trigger event that will occur whenever the ADC level is greater than the value specified
ADC Trigger Level Set and read the ADC trigger level.
Image Reject Traces Enabled Set and read the show or hide state of the image reject traces in the measurement parameters dialog.
LO Feed Thru Monitor Enabled Set and read the LO feedthru monitoring state.
LO Feed Thru Monitor Receiver Set and read the target receiver for LO Feedthru monitor. Select the reference receiver in most cases. Default "a1".
LO Feed Thru Monitor Tolerance Set and read the threshold level for warning. Default -40.
LO Feed Thru Monitoring Type Set and read the monitoring method for LO feedthru monitor. Default Band power.
Multitone Data Display Set and read the data display mode. Default All.
Multitone Harmonics Reject Count Set and read the number of test signal harmonics you want to be protected against. This adds constraints to the list of LOs used to cover the span. Default 0.
Multitone Image Rejection Enabled Enables or disables multitone image rejection.
Multitone Nyquist Protection Level Sets and returns the Nyquist protection level. Default 0.
Multitone Offset Frequency Set and read the multitone image rejection offset frequency. Default 0.
Multitone Repitition Rate Set and read the test signal repetition rate (in seconds). Default 1E6.
Multitone Spacing Set and read the tone spacing of the multitone signal (in Hz). Default 1000000.
Phase Computing Enabled Enables or disables phase computing.
Phase Display Level Set and read the phase display minimum level in dBm. Default -60 dBm.
Phase Stitching Common Enabled Enables or disables selected receiver, or receiver with highest energy in the overlap area, to drive the phase stitching for all receivers.
Phase Stitching Enabled Enable or disable phase stitching.
Phase Stitching HW Timestamp Enabled Enables or disables phase stitching using hardware time stamps to determine the phase of each LO acquisition and stitch them back together with phase consistency.
Phase Stitching Level Set and read the minimum tone power level to perform phase stitching in dBm. Default -150 dBm.
Phase Stitching Receiver Assigns a specified receiver for phase stitching. Default R1.
Phase Stitching Receiver Auto Enabled Set and read the receiver for stitching automatically.
Pulse Search Count Set and read the maximum pulse search count. Default 100.
Pulse Search Duty Cycle Tolerance Set and read the duty cycle tolerance.
Pulse Search Period Set and read the period starting point in seconds.
Pulse Search Period Tolerance Set and read the period tolerance. Only solutions that ensure this tolerance will be accepted.
Pulse Search Priority Set and read the priority choice for the search algorithm. Default Width.
Pulse Search Width Set and read the width starting point in seconds.
Pulse Search Width Tolerance Set and read the width tolerance. Only solutions that ensure this tolerance will be accepted.
Trigger Periodic Counter Enabled Set and read the ON or OFF state of a measurement trigger event based on the specified period.
Trigger Periodic Counter Value Set and read the periodic counter value. This command initiates a measurement trigger event based on the specified period.
Vector Averaging Enabled Set and read the ON or OFF state of the vector averaging.
Vector Averaging Value Set and read the vector averaging value.
