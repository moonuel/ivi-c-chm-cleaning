Attributes

Name Description
Display Select Select the range to be displayed on the VNA x-axis. All traces in the channel have this same x-axis scaling. Default "Receivers".
Range Coupled Sets and returns the state of coupling (ON or OFF) of the specified range to the primary range.
Range Frequency Cw Sets and returns the CW frequency (in Hz).
Range Frequency Divisor Sets and returns the divisor value. Default 1.
Range Frequency Multiplier Sets and returns the multiplier value to be used when coupling this range to the primary range. Default 1.
Range Frequency Offset Sets and returns the offset value to be used when coupling this range to the primary range. Default 0.
Range Frequency Start Sets and returns the Start value of frequency range (in Hz). Default - Minimum frequency of the VNA.
Range Frequency Stop Sets and returns the Stop value of frequency range (in Hz). Default - Maximum frequency of the VNA.
Range No Range number. If unspecified, value is set to 1.
Range Sweep Type Sets and returns the sweep type to be used with the specified range. Default Linear
State Turns Frequency Offset ON and OFF. Frequency offset settings are not enabled until this setting is ON.
