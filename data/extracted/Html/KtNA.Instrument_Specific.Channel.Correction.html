Attributes

Name Description
AsyncSweepingEnabled This command enables/disables asynchronous sweeping during a calibration acquisition. Default 1.
CacheMode Set this mode at ON for SNP file fast saving. Default 1.
CalMethod For UNGUIDED calibration, sets the calibration method (also known as 'Calibration Type' on calibration dialog box.)
ENRNoiseSourceSerialNo Sets and returns the serial number of noise source.
ENRTableID Sets and returns ID of ENR table.
ErrorTermModel Sets and returns the error term model. Default Term10.
ForwardCalEnabled Sets the direction a calibration will be performed when only one set of standards is used. Default 1.
GCCalKit Set and read the cal kit to be used for calibrating at the port 1 reference plane when the power sensor connector is different from the DUT port 1.
GCConnectorType Set and read the power sensor connector type which is used to perform the Source Power Cal portion of a Gain Compression Calibration.
GCPowerLevel Set and read the power level at which to perform the Source Power Cal portion of a Gain Compression (Opt S93086A/B) Calibration. Default 0.
InterpolationEnabled Turns correction interpolation ON or OFF. Default 1.
IsolAveragingIncrement Specifies amount to increment (increase) the channel averaging factor during isolation measurement of the ECal module during an unguided ECal calibration. Default 8.
IsolEnabled Specifies whether or not the isolation state of the ECal module will be measured as part of an unguided ECal calibration. Default 0.
NoiseSourceTemp Sets and returns the temperature of the noise source connector.
ReceiverPowerOffset Adjusts a receiver power cal to account for components or adapters that are added between the source port and receiver while performing this cal. Default 0.
SystemImpedance Sets and returns the system impedance value for the analyzer. Default 50.
TwoStandardsEnabled Specifies the acquisition of calibration data using ONE or TWO sets of standards. Default 1.
VelocityFactor Sets the velocity factor to be used with Electrical Delay and Port Extensions. Default 1.
WaveCorrMethod Sets and returns the wave correction method. Default MATC.
WaveNormMethod Sets and returns the wave normalization method. Deafult POW.
