Attributes

Name Description
HoldOffTime Set and read the holdoff time for Video Trigger in Narrow band. Default 1.
Hysteresis Set and read the hysteresis value (frequency) for video trigger in Narrow band. Default 1E3.

Threshold Set and read the minimum power value which disables the video trigger in Narrow band.- Default -20dB.
