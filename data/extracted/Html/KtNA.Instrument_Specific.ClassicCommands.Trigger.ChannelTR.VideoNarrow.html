Attributes

Name Description
FrequencyHysteresis Set and read the hysteresis value (frequency) for video trigger in Narrow band. Default 1E3.
FrequencyTarget Set and read the target frequency value which make a video trigger in Narrow band. Default 1E6.
Hoff Set and read the holdoff time for Video Trigger in Narrow band. Default 1.
NarrowBandNo Sets and Gets Transient narrow band number. Choose from 1 or 2.
PowerHysteresis Set and read the hysteresis value (Power) for video trigger in Narrow band. Default 10.
PowerTarget Set and read the target power value which make a video trigger in Narrow band. Default 0.
Slope Set and read the trigger edge from positive or negative for video trigger in Narrow band. Default Positive.
Threshold Set and read the minimum power value which disables the video trigger in Narrow band.- Default -20dB.
