Attributes

Name Description
CcheckParameter Specifies an existing measurement to be used for the Confidence Check.
CSet Name Sets or queries the name of the Cal Set currently applied to the specified channel.
ENRCalTableIDData Sets and returns ID of ENR table.
ENRCalTableSerialData Sets and returns the serial number of noise source.
GCSetupPower Set and read the power level at which to perform the Source Power Cal portion of a Gain Compression (Opt S93086A/B) Calibration. Default 0.
GCSetupSensorCkit Set and read the cal kit to be used for calibrating at the port 1 reference plane when the power sensor connector is different from the DUT port 1.
GCSetupSensorConnector Set and read the power sensor connector type which is used to perform the Source Power Cal portion of a Gain Compression Calibration.
ImpedanceInputMagnitude Sets and returns the system impedance value for the analyzer. Default 50.
MethodsMatch Turns match-correction ON or OFF. Use this command AFTER performing an Guided Power Cal. Default 1.
Model Sets and returns the error term model. Default Term10.
RpowerOffset Adjusts a receiver power cal to account for components or adapters that are added between the source port and receiver while performing this cal. Default 0.
RVelocityCoax Sets the velocity factor to be used with Electrical Delay and Port Extensions. Default 1.
SforwardState Sets the direction a calibration will be performed when only one set of standards is used. Default 1.
State Turns error correction ON and OFF for the specified channel.
TcoldUserValue Sets and returns the temperature of the noise source connector.
TstandardsState Specifies the acquisition of calibration data using ONE or TWO sets of standards. Default 1.
WaveMethod Sets and returns the wave correction method. Default MATC.
WaveNormMethod Sets and returns the wave normalization method. Deafult POW.
