Attributes

Name Description
ALC Mode Sets and returns the ALC mode for the specified channel and port. Default Internal.
Attenuation Level Sets or returns the attenuation level for the selected channel.
Center Sets or returns the power sweep center power.
Couple Turns Port Power Coupling ON or OFF.
Level Sets or returns the RF power output level in dBm.
Mode Sets the state of VNA source for the specified port.
Port No Source port number of the VNA. Choose any number between 1 and 50. Default 1.
Port Start Sets and reads the power sweep start power value in dBm for a specific port. This allows uncoupled forward and reverse power sweep ranges.
Port Stop Sets and reads the power sweep stop power value in dBm for a specific port. This allows uncoupled forward and reverse power sweep ranges.
Slope Sets the RF power slope value in in db/GHz.
Slope State Turns Power Slope ON or OFF.
Span Sets the power sweep span power in dBm.
Start Sets or returns the power sweep start power for ALL ports being used by the specified channel.
Stop Sets or returns the power sweep stop power for ALL ports being used by the specified channel.
