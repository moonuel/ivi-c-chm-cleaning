IVI-C Programming Basics

The topics in this section explain how to perform basic programming tasks:

  • Initializing the IVI-C Driver provides information on the basic use and parameters of the Initialize functions. Read this section first, then go to the appropriate section in ADE Programming for details on ADE-specific syntax.
  • Programming the Instrument explains how to access the functionality of the instrument.
  • Simulating an Instrument explains how to use the driver without connecting to an actual instrument.

Additional topics are included in this section if your driver is class-compliant or supports repeated capabilities.
