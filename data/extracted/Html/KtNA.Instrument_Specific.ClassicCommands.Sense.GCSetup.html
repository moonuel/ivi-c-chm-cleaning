Attributes

Name Description
Amode Set and read the method by which gain compression data is acquired. Default SmartSweep
Compression Algorithm Set and read the algorithm method used to compute gain compression. Default LinearGain.
Compression Backoff Level Set and read value for the BackOff compression algorithm in dB.
Compression Delta X Set and read the 'X' value in the delta X or Y compression algorithm.
Compression Delta Y Set and read the 'Y' value in the delta X or Y compression algorithm.
Compression Interpolate Sets whether or not interpolation should be performed on 2D measured compression data.
Compression Level Set and read the desired gain reduction (from reference gain).
Compression Phase Level Set and read the desired phase to measure compression.
Compression Phase Mode Set and read compression format to be either magnitude, phase, or magnitude and phase. Default Magnitude.
Compression Saturation Level Set and read the deviation dB from the maximum Pout. This is the point of saturation.
Eos Operation This setting is used to protect a sensitive device from too much power during the sweep retrace. Default Standard.
Pmap Source Override This function is used when GCX channels with a reference mixer and one GCA channel without a reference mixer are set up simultaneously and you want to use the same path configuration for all channels. This remaps the source port to Port 1.
Power Linear Input Compute Aperture Set and read the aperture to use when computing the linear input power.
Power Linear Input Level Set and read the input power at which Linear Gain and all S-parameters are measured.
Power Reverse Level Set and read the reverse power level to the DUT.
Power Start Level Set and read the start power level.
Power Stop Level Set and read the stop power level.
Safe Cp Adjustment Set and read the Safe Sweep COARSE power adjustment in dBm.
Safe Dc Mlimit Set and read the maximum limit of the external DC device. Default -5.
Safe Dc Parameter Set and read the name of the external DC device.
Safe Enable Set and read the (ON or OFF) state of Safe Sweep mode.
Safe Fp Adjustment Set and read the Safe Sweep FINE power adjustment in dBm.
Safe Mlimit When the VNA port that is connected to the DUT Output measures this value, the input power to the DUT is no longer incremented at that frequency.
Safe Threshold Set and read the compression level in which Safe Sweep changes from the COARSE power adjustment to the FINE power adjustment.
Smart Cdc Set and read the DC readings at the compression point in the last iteration of a smart sweep.
Smart Miterations Set and read the maximum permitted number of iterations which SMART Sweep may utilize to find the desired compression level, to within the specified tolerance.
Smart Siterations Set and read enable for showing intermediate results for each iteration of SMART Sweep
Smart Stime Set and read the amount of time SMART Sweep will dwell at the first point where the input power changes by the Backoff or X level.
Smart Tolerance Set and read the acceptable range SMART Sweep will allow for the measured compression level.
Sweep Frequency Points Set and read the number of data points in each frequency sweep.
Sweep Power Points Set and read the number of data points in each power sweep. Default 21.
Sweep Power Smooth Set and read the state of the power smoothing.
Sweep Power Smooth Aperture Set and read the power smoothing aperture in percent. Default 25.
