Attributes

Name Description
AcquisitionMode Sets all ports to pre-sweep or point leveling mode for the specified channel. Default PreSweep.
Fast Sets and returns the state of a separate IFBW setting for leveling sweeps. Default 1.
Ftype Sets and returns the frequency range to use for receiver leveling. Default Auto.
IFBW Sets and returns the IFBW to be used for leveling sweeps. Default 100 kHz.
IterationEnable Enables or disables the receiver leveling maximum iteration search function set. Default 1.
IterationValue Sets and returns the maximum iterations to be used in order to achieve the tolerance setting. Default 10.
LSPC Sets and returns the state of Use Last Result for Source Power Cal. Default 0.
ModulationApertureOffset Sets and returns the offset of the frequency aperture used to measure the signal power. Default 0.
ModulationApertureSpan Sets and returns the span of the frequency aperture used to measure the signal power. Default 10e6.
ModulationApertureState Sets and returns the state of the receiver leveling aperture settings for the specified source for Modulation Distortion measurements. Default 0.
ModulationBandwidthNoise Sets and returns the receiver noise bandwidth value for the specified source for Modulation Distortion measurements. Default 1000.
Offset Sets and returns the offset of the frequency aperture used to measure the signal power. Default 0.
Reference Sets and returns the reference receiver to be used with Receiver Leveling.
SafeMax Sets and returns the maximum power level for Safe Mode. Default 30 dB.
SafeMin Sets and returns the minimum power level for Safe Mode. Default -95 dB.
SafeState Sets and returns the state of Safe Mode. Deafult 0.
SafeStep Sets and returns the maximum step power level for Safe Mode. Default 1 dB.
State Sets and returns the state of Receiver Leveling for the specified source port. Default 0.
Tolerance Sets and returns the tolerance value for leveling sweeps. Default 1 dB.
