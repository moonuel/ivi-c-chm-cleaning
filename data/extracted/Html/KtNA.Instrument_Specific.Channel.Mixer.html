Attributes

Name Description
Absolute Phase Enabled Sets and returns the state of SMC Phase measurements to use absolute phase
ELO Enabled Sets and returns the ON or OFF state of Embedded LO.
IF Freq Sideband When two LO stages are used, sets or returns whether to select the sum or difference for the IF1 product.
IF Freq Start Sets or returns the IF start frequency value of the mixer.
IF Freq Stop Sets or returns the stop frequency value of the mixer IF frequency.
Input Denominator Sets or returns the denominator value of the Input Fractional Multiplier.
Input Fixed Frequency Sets or returns the fixed frequency of the input.
Input Numerator Sets or returns the numerator value of the Input Fractional Multiplier.
Input Power Sets or returns the value of the Input Power.
Input Start Frequency Sets or returns the Input start frequency value of the mixer.
Input Start Power Sets the input start power for a power sweep in a mixer channel like SMC. The value is only used when the sweep type is power sweep.
Input Stop Frequency Sets or returns the Input stop frequency value of the mixer.
Input Stop Power Sets the input stop power for a power sweep in a mixer channel . The value is only used when the sweep type is power sweep.
Input Sweep Mode Sets or returns the Input sweep mode. Default Fixed.
Nominal Incident Power Enabled Toggles the Use Nominal Incident Power setting ON and OFF. This setting is ONLY to be used with SMC measurements.
Normalization Point Sets or returns the data point for normalizing the phase measurement.
Output Fixed Frequency Sets or returns the output fixed frequency of the mixer.
Output Freq Sideband Specify whether to select the sum (High) or difference (Low) products.
Output Start Frequency Sets or returns the Output start frequency of the mixer.
Output Stop Frequency Sets or returns the Output stop frequency of the mixer.
Output Sweep Mode Sets or returns the Output sweep mode. Default Fixed.
Phase Enabled Sets and returns the state of SMC Phase measurements and calibrations.
Stage Count Number of IF stages (LOs) used in the mixer.
X Param Sets or returns the swept frequency range to display on the X-axis for the IMDx or NFx channel. Default Output.
