Name Description

Frequency Fixed Sets or returns the fixed frequency of the specified mixer LO.
Frequency Ilti Specifies whether to use the Input frequency that is greater than the LO or less than the LO.
Frequency Mode Sets or returns the LO sweep mode. Default Fixed.
Frequency Start Sets or returns the LO start frequency value.
Frequency Stop Sets or returns the LO stop frequency value.
Name ets or returns the name of the VNA internal source or external source to use as the LO in a converter measurement.
Numerator Sets or returns the numerator value of the LO Fractional Multiplier.
Power Sets or returns the LO Power fixed value.
Power Start For an LO power sweep, sets or returns the LO power start value. Default -20 dBm.
Power Stop For an LO power sweep, sets or returns the LO power stop value. Default -10 dBm.
Stage No LO stage number. Choose 1 or 2.
