KtNA_ChannelVCOSetDCVoltage Function
VCO See Also

Set the DC voltage level after sweep.

Function Tree Node: \KtNA\Instrument Specific\Channel\VCO\SetDCVoltage
Declaration: KtNA.h
Implementation: KtNA.dll

Syntax

Visual C++
ViStatus KtNA_ChannelVCOSetDCVoltage(
	ViSession Vi,
	ViConstString ChannelName,
	ViInt32 Device,
	ViInt32 Voltage
);

Parameters

Vi

The ViSession handle that you obtain from the KtNA_init or KtNA_InitWithOptions function.The handle identifies a particular instrument session.

ChannelName

The physical or virtual repeated capability identifier. Pass VI_NULL or empty string if operation does not apply to a repeated capability.For valid values, see the Channel repeated capability.

Device

DC device. Choose from VControl, VSupply1 or VSupply2.

Voltage

DC voltage level after sweep. Default Start.

Return Value


Success or failure code.

Instrument Command

SENSe:VCO:SOURce:DC:AFTer:STATe "{}" , {STAR|STOP|USER}
