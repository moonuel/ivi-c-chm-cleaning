Name Description

File Erase Enables or disables erasing output data files after each sweep.
File Prefix Set and return the file name prefix for the data file.
File Text Enables or disables text file (.txt) output. Data is not exported until the next new sweep occurs.

File Text Markers All Enables or disables adding all marker data to the text file (.txt) output.
File Text Verbose Enables or disables exporting frequency and data for text files.
Iq Aligned Enable or disable recomputing the advanced IQ parameters (IQ center, IQ sample rate, IQ number of points) automatically if a SA sweep parameter change is performed.
Iq Carrier Frequency Set and read the IQ carrier frequency.
Iq File Binary Enables or disables binary file (.bin) output. Data is not exported until the next new sweep occurs.

Iq Sample Rate Set and read the IQ sample rate.

Keep State Set and read the ON or OFF state of the end of sweep processing.
