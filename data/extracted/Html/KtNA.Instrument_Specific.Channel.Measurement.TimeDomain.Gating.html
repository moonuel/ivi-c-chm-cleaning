Attributes

Name Description
Center Sets or returns the gate filter center time in seconds.
Enabled Turns gating state ON or OFF.
FilterShape Sets or returns the gating filter shape when in time domain.
FilterType Sets or returns the type of gate filter used.
ParametersToCouple Specifies the time domain gating parameters to be coupled. The settings for those parameters will be copied from the selected measurement to all other measurements on the channel.
Span Sets or returns the gate filter span time in seconds.
Start Sets or returns the gate filter start time in seconds.
Stop Sets or returns the gate filter stop time in seconds.
