Name Description

SegmentNo Segment number, if unspecified, value is set to 1.

SegmentXStop Sets the stop (end) of the X-axis stimulus value.
SegmentYStart Sets the start (beginning) of the Y-axis amplitude (response) value.
SegmentYStop Sets the stop (end) of the Y-axis amplitude (response) value.
TestingFailSoundEnabled Turns limit testing fail sound ON or OFF.
