Tree Node: \KtNA\Instrument Specific\Display\Color

Functions

Name Description
Get Background Return the background color for the inactive windows on the VNA display or hardcopy print.
Get Background For Active Return the background color for the active window on the VNA display or hardcopy print.
Get Display Color No Gets the value of the DISPLAY_COLOR_NO attribute.
Get Display Color Trace Pen No Gets the value of the DISPLAY_COLOR_TRACE_PEN_NO attribute.
Get Failed Trace Limit Line Return the limit line color of failed traces or failure indicators (dots) and the word Fail.
Get Grid Return the inner lines of all grid in all windows, and the grid frame in inactive windows for the VNA display or hardcopy print.
Get Grid For Active Return the labels and grid frame colors in the active window for the VNA display or hardcopy print.
Get Memory Trace Return the memory trace color for nth trace in a window.
Get Memory Trace Marker Return the color of memory trace markers for nth trace in a window.
Get Trace Data Return the color of Data and Limit Lines for nth trace in a window.
Get Trace Marker Return the color of data trace markers for nth trace in a window.
Get Window Label Return the Inactive (not selected) Window Labels for the VNA display or hardcopy print.
Load Load a color theme from a disc file.
Reset Resets the current theme to the default VNA colors.
Set Background Set the background color for the active window on the VNA display or hardcopy print. Default Display = 0,0,24 (Black), Print = 255,255,255 (White).
Set Background For Active Set the background color for the inactive window on the VNA display or hardcopy print. Default Display = 0,0,0 (Black), Print = 255,255,255 (White).
Set Display Color No Sets the value of the DISPLAY_COLOR_NO attribute.
Set Display Color Trace Pen No Sets the value of the DISPLAY_COLOR_TRACE_PEN_NO attribute.
Set Failed Trace Limit Line Set the limit line color of failed traces or failure indicators (dots) and the word Fail. Default Display = 255,20,20, Print = 255,20,20.
Set Grid Set the inner lines of all grid in all windows, and the grid frame in inactive windows for the VNA display or hardcopy print. Default Display = 100,100,100, Print = 50,50,50.
Set Grid For Active Set the labels and grid frame colors in the active window for the VNA display or hardcopy print. Default Display = 175,175,175, Print = 0,0,0 (Black).
Set Memory Trace Set the memory trace color for nth trace in a window. Default Varies for each trace.
Set Memory Trace Marker Set the color of memory trace markers for nth trace in a window. Default Varies for each trace.
Set Trace Data Set the color of Data and Limit Lines for nth trace in a window. Default Varies for each trace.
Set Trace Marker Set the color of data trace markers for nth trace in a window. Default Varies for each trace.
Set Window Label Set the Inactive (not selected) Window Labels for the VNA display or hardcopy print. Default Display = 160,160,160, Print = 0,0,0 (Black).
Store Saves the current color theme to a disc file.
