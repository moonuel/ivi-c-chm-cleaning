IVI-C Programming Guide and KtNA Driver Reference  
Welcome to KtNA IVI-C Programming

Welcome to programming with the KtNA v. 3.0.8 IVI-C driver. Keysight's IVI drivers simplify the creation and maintenance of instrument control applications, give you programmatic control of your instrumentation, and enable instrument interchangeability and code reuse.

The KtNA driver is a fully compliant IVI-C driver that supports compiling application programs for 32- or 64-bit platforms.

About this Guide

This guide is for users who are writing programs for the KtNA driver. It provides programming information and an extensive reference in two main sections:

  • IVI-C Programming Guide describes programming tasks using the driver's IVI-C functionality:
    • IVI-C Programming Basics explains how to initialize the driver, access repeated capabilities and use simulation
    • ADE Programming provides additional information specific to using the driver in Visual C++, MATLAB, VBA/Excel, LabWindow/CVI, LabVIEW, and VEE Pro
    • Using Driver Source Code describes how to make basic modifications to the IVI-C driver source code
    • About IVI provides information about the IVI Foundation and specifications that is helpful to developers new to IVI programming
  • KtNA Driver Reference provides descriptions and reference documentation for each IVI.NET method and property:
    • KtNA shows a hierarchical view of all KtNA IVI-C functions and attributes
    • Functions and Attributes lists all IVI-C functions and attributes alphabetically
    • Functions lists all IVI-C functions alphabetically and contains documentation for each function
    • Attributes lists all IVI-C attributes alphabetically and contains documentations for each attribute
    • Errors and Warnings lists all driver errors and warnings ordered by error code
    • Instrument Model Support describes any methods and properties that are not be supported by all instrument models
    • Status System Overview describes the instrument status register structure
    • Instrument Command Reference provides a table associating instrument command strings with driver methods and properties
    • IVI Compliance Information provides IVI-required compliance information for the KtNA driver, including class information

Example Programs

Example programs that demonstrate basic driver usage in a variety of ADEs are installed with this driver. You can build and run these programs. To access them, go to Windows Start Menu > All Programs (apps) > Keysight Instrument Drivers > IVI-C KtNA 3.0.8.
