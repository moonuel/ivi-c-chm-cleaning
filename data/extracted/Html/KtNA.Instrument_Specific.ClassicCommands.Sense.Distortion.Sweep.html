Attributes

Name Description
Carrier Frequency Sets and returns the carrier center frequency. Default 1.5GHz.
Dwell Sets and returns the delay in seconds before the test signal to allow the RF source to settle. Default 0s.
Power Sparam Level Sets and returns the power level used to measure the DUT S-parameters when operating in its linear region. Default -30dBm.
Sparam Bwidth Sets and returns the IF bandwidth for the linear S21 sweep. Default 1 kHz
Sparam Reuse Enable or disable the re-use of the previous linear S-parameter measurements if available and skip sweep.
Sparam State Enable or disable S-parameter sweep. If disabled, curve fitting (smoothing) is used instead of the S-parameter sweep.
Sparam Step Sets and returns the step size for the linear S21 sweep. Default 1 MHz.
Type Selects between a fixed or power sweep type of sweep. Default Fixed.
