KtNA_CalSetsCharacterizeFixture Function
CalSets See Also

Characterizes a fixture based on two Cal Sets.

Function Tree Node: \KtNA\Instrument Specific\CalSets\Characterize Fixture
Declaration: KtNA.h
Implementation: KtNA.dll

Syntax

Visual C++
ViStatus KtNA_CalSetsCharacterizeFixture(
	ViSession Vi,
	ViConstString Calset1,
	ViConstString Calset2,
	ViInt32 PortNo,
	ViConstString CharacterizationFile,
	ViInt32 Format
);

Parameters

Vi

The ViSession handle that you obtain from the KtNA_init or KtNA_InitWithOptions function.The handle identifies a particular instrument session.

Calset1

Name of an existing Cal Set 1 which describes the cal closest to the VNA. The Cal Set must reside on the VNA.

Calset2

Name of an existing Cal Set 2 which describes the cal closest to the DUT. The Cal Set must reside on the VNA.

PortNo

Port number described in the Cal Sets.

CharacterizationFile

Name of the S2P file containing the adapter/fixture characterization.

Format

Data format for the resulting s2p file

Return Value


Success or failure code.

Remarks

The stimulus settings of the two Cal Sets do NOT have to be identical, but they MUST have a common frequency range for interpolation. A new S2P file is created.

Instrument Command

CSET:FIXTure:CHARacterize "{Calset1}" , "{Calset2}" , {PortNo} , "{CharacterizationFile}" , {REIM|LOG|LIN}
