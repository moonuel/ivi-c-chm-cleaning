Attributes

Name Description
Dut Input Sets and returns the VNA port number connected to the DUT input. Default 1.
Dut Nominal Gain Sets and returns the nominal DUT gain. Default 0.
Dut Nominal Nf Sets and returns the nominal DUT noise figure. This value is used by the EVM measurement. Default 0.
Dut Nominal Nf Include Include or exclude the nominal DUT noise figure value. Default 0.
Dut Output Sets and returns the VNA port number connected to the DUT output. Default 2.
Source Attenuation Include Enables or disables the external modulated source to pass through the attenuator of the specified PNA source and the attenuator setting is used to calculate the power levels.
Source Nominal Amplifier Sets and returns the nominal gain of an external source amplifier. Default 0.
