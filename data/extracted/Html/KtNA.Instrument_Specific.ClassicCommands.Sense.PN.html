Attributes

Name Description
AdjustConfigureFrequencySearch Enables and disables a broadband carrier search within the range specified. Default 1 (On).
AdjustConfigureLevelThreshold Sets and returns the threshold to use during a carrier search. Default -20dBm.
AttenuationAuto Enable and disable the automatic attenuation control. Default 1 (On).
AttenuationValue Sets the attenuation level for the selected channel. The range is from 0 to 30 dB with the step size 1 dB. Default 10.
BasebandGain Sets and returns the setting for baseband gain. Default 0.
BasebandInputCoupling Sets and returns the setting for baseband input. Default ACLF.
BwidthRatio Sets and returns the resolution bandwidth ratio, which is the specified resolution bandwidth percentage of every half decade offset frequency. Default 10.
CompositionModel Choose decomposition model of AM/PM composite measurement range. Default 4.
CompositionRange Choose the range of AM/PM composite measurement. Default R30.
CorrelationCount Sets and returns the cross correlation count number. Correlation count number from 1 to 100,000. Default 1.
FaverageFactor Sets and returns the FFT average factor number. FFT average factor number from 1 to 10,000. Default 1.
Ntype Sets and returns the noise type to phase, residual or baseband noise. Default Phase.
PulsePeriod Set or returns Pulse period for phase noise measurement. Default 100E-6.
PulseState Enables or disables Pulse setup for phase noise measurement. Default 0 (Off).
SweepCarrierFrequency Sets and returns the carrier frequency. Default 1 GHz.
SweepNoiseMode Sets and returns the sweep noise mode. Default Normal.
SweepOffsetExtend Sets and returns the Offset frequency extension status. Default 0 (Off).
SweepType Sets and returns the phase noise sweep type to logarithmic or segment.
Xgindicator This command turns the cross-correlation gain indicator on and off. Default 0 (Off).
