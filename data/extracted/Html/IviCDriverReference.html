KtNA Driver Reference

This section documents each function and attribute exposed from the KtNA IVI-C driver. Note that the first two topics show the same information in a hierachical view and a flat view. A list of instrument command strings and their associated driver methods and properties is also provided. See the following topics:

  • KtNA shows a hierarchical view of all KtNA IVI-C functions and attributes
  • Functions and Attributes lists all IVI-C functions and attributes alphabetically
  • Functions lists all IVI-C functions alphabetically and contains documentation for each function
  • Attributes lists all IVI-C attributes alphabetically and contains documentation for each attribute
  • Errors and Warnings lists all driver errors and warnings ordered by error code
  • Instrument Model Support includes methods and properties that are not supported by particular instrument models
  • IVI Compliance Information — KtNA information (including instrument class information) required for IVI compliance
