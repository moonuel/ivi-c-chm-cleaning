Attributes

Name Description
Cut Off Frequency Set and read the ADC filter cutoff frequency.
Dither Enabled Set and read the ON or OFF state of the FFT grid dither setting
Filter Auto Enabled Set and read how the ADC filter is set. When ON, the ADC filter is set based on the start and stop frequencies and the ADC sampling frequency.
FIR Filter Enabled Set and read the FIR filter for 25 MHz decimation.
Force Record Size Value Set and read the integer size value for the force ADC record size feature.
Multiple Recording Enabled Set and read the ON or OFF state of the multiple recording function.
Overload Threshold Set and read the ADC overload threshold percentage. Default 80.
Record Chunk Size Set and read the size of the ADC record chunks.
Record Chunk Wait Period Set and read the period to wait between ADC record chunks.
Record Size Manual Enabled Set and read the ADC record size mode.
Sample Rate Set and read the IQ sample rate.
Sample Rate Auto Enabled Set and read the ADC sample rate mode.
Sample Stacking Enabled Set and read the ON or OFF state of the ADC sample stacking.
Stacking Value Set and read the ADC stacking value.
