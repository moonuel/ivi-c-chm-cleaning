Attributes

Name Description
Baseband Enabled Enables and disables baseband sweep independent of the LO sweep to allow signals down to 1 Hz to be analyzed.
BW Ratio Set and read the RBW or VBW ratio. Default 1.0
Detector Bypass Enabled Set and read the ON or OFF state of the detector bypass setting.
Detector Function Set and read the detector type. Default Peak.
DFT Bandwidth Auto Enabled Set and read the default values for DFT bandwidth.
DFT Record Size Type Set and return the DFT record size type. Default Radix.
FFT Dither Enabled Set and read the ON or OFF state of the FFT grid dither setting
ForceLO Divider Set and read the divider in the equation for setting LO frequency. Default 1.
ForceLO Enabled Set and read enable force LO to frequency mode.
ForceLO Multiplier Set and read the multiplier in the equation for setting LO frequency. Default1.
ForceLO Source Set and read the source name in the equation for setting LO frequency.
Freq Converter Dual Band Config Enabled Set and read the ON on OFF state of the dual band configuration for fast setting of the frequency converter DUT.
Freq Converter Input Center Freq Set and read the frequency converter DUT input center frequency in Hz.
Freq Converter Input Port Set and read the port number of the VNA connected to the input of the frequency converter DUT. Default 0.
Freq Converter Input Source Set and read the source name that generates the modulated signal. Default "None".
Freq Converter Inverting Enabled Enable or disable inverting the spectrum (or mirror) by the frequency converter operation.
Freq Converter LO Offset Set and read the LO offset value in Hz. Default 0.
Freq Converter Output Center Freq Set and read the frequency converter DUT output center frequency in Hz.
Freq Converter Output Port Set and read the port number of the VNA connected to the output of the frequency converter DUT. Default 0.
Frequency Span Set and read the Frequency Span or RBW ratio.
Image Reject Mode Set and read the image reject mode. Default Normal.
Image Rejection Strength Sets and returns the image rejection strength. Default Medium
LO Force Frequency Set and read enable force LO to the specified frequency in Hz.
LO Randomize Enabled Set and read the LO randomize state.
MaxDFT NarrowBW Set and read the maximum value for narrow DFT bandwidth.
MaxDFT WideBW Set and read the maximum value for wide DFT bandwidth.
MinDFT NarrowBW Set and read the minimum value for narrow DFT bandwidth.
MinDFT WideBW Set and read the minimum value for wide DFT bandwidth.
OccBW Min Search Freq Set and read the minimum search frequency to use during an Occupied BW search measurement. Default 250MHz.
Resolution Bw Set and read the resolution bandwidth.
ResolutionBW Auto Enabled Set and read how the resolution bandwidth is set. When ON, the resolution bandwidth is set based on Span or RBW ratio.
ResolutionBW Shape Set and read the resolution bandwidth shape. Default Gaussian.
VideoBW Set and read the video bandwidth.
VideoBW Auto Enabled Set and read how the video bandwidth is set. When ON, video bandwidth is set based on RBW or VBW ratio.
VideoBW Averaging Type Set and read the averaging type. Default Power.
