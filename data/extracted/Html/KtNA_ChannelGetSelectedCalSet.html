KtNA_ChannelGetSelectedCalSet Function
Channel See Also

Returns the name of the Cal Set that is applied to the specified channel.

Function Tree Node: \KtNA\Instrument Specific\Channel\Get Selected CalSet
Declaration: KtNA.h
Implementation: KtNA.dll

Syntax

Visual C++
ViStatus KtNA_ChannelGetSelectedCalSet(
	ViSession Vi,
	ViConstString ChannelName,
	ViInt32 Mode,
	ViInt32 ValBufferSize,
	ViChar[] Val
);

Parameters

Vi

The ViSession handle that you obtain from the KtNA_init or KtNA_InitWithOptions function.The handle identifies a particular instrument session.

ChannelName

The physical or virtual repeated capability identifier. Pass VI_NULL or empty string if operation does not apply to a repeated capability.For valid values, see the Channel repeated capability.

Mode

Choose from GUID or Name to specify which string is returned. Default GUID.

ValBufferSize

Number of elements in Val.

Val

Name or GUID of the applied Cal Set. If no Cal Set is applied to the specified channel, then "No Calset Selected" is returned.

Return Value


Success or failure code.

Instrument Command

SENSe{Channel}:CORRection:CSET:ACTivate? {GUID|NAME}

Requirements

Supported Models

All supported

Defined Values For Parameter Mode

Name Value Description
KTNA_VAL_CAL_SET_MODE_GUID 0 GUID of the Cal Set is returned. (Default setting)
KTNA_VAL_CAL_SET_MODE_NAME 1 Name of the Cal Set is returned.
