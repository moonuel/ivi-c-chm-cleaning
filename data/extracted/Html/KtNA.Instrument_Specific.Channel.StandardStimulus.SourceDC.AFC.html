Attributes

Name Description
Enabled Enable and disable the Auto Tune VControl voltage. Default 0 (Off).
FrequencyBand Selects the frequency band which include the target VCO frequency. Default R8G
MaxLimit Set and read the maximum limit of VControl during Auto Tune sequence. Default 18.
MaxNoOfIteration Set and read the maximum number of readings to take at each data point for iterating the frequency. Default 10.
MinLimit Set and read the minimum limit of VControl during Auto Tune sequence. Default 0.5.
Sensitivity Set and read the sensitivity of VCO (Hz/Volt). This value is used at the first loop of sequence. Default 50.
TargetFrequency Set and read the target frequency. Default 1E6.
Tolerance Set and read the maximum desired deviation from the specified target frequency. Default 1E5.
