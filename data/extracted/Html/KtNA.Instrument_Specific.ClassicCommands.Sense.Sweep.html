Attributes

Name Description
Dwell Sets or returns the dwell time in seconds between each sweep point.
Dwell Auto Specifies whether or not to automatically calculate and set the minimum possible dwell time.
Dwell SDelay Sets or returns the time in seconds to wait just before acquisition begins for each sweep.
Generation Sets sweep as Stepped or Analog.
Generation Point Sweep Turns ON and OFF point sweep mode.
Groups Count Sets the trigger count (groups) for the specified channel. Set trigger mode to group after setting this count.
LF Extension State Turns ON or OFF low frequency extension for extending the range down to 1 kHz start frequency.
Mode Sets the number of trigger signals the specified channel will ACCEPT.
Points Sets or returns the number of data points for the measurement.
SLocal Maximum Sets the Shift LO maximum frequency for the selected channel.
SLocal State Turns ON or OFF the Shift LO mode for the selected channel.
Speed Sets and returns the state of Fast Sweep mode.
Step Sets the frequency step size in Hz across the selected frequency range. This effectively sets the number of data points.
Time Sets or Returns the time in seconds the analyzer takes to complete one sweep.
Time Auto Turns the automatic sweep time function ON or OFF.
Trigger Delay Sets and reads the trigger delay in seconds for all measurements in the specified CHANNEL.
Trigger Mode Sets and reads the trigger mode for the specified channel. This determines what EACH signal will trigger.
Type FACW Enables Fast CW sweep and sets the number of data points for the channel.
