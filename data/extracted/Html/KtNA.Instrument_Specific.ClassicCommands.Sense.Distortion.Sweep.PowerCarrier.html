Attributes

Name Description
Level Sets and returns the carrier power level used for the distortion test at either the input or output of the DUT. Default -10dBm.
Level Port Set and read where to apply power to the DUT (input or output). Default DIN1.
Level Type Selects between a ramp type power sweep or a list of power values to define a power sweep. Default Ramp.
List Level Sets and returns the power level used for the specified row in the power sweep list table. Default -10 dBm.
List Nbw Sets and returns the noise bandwidth setting used for the specified row in the power sweep list table. Default 100Hz.
List Nbw Mode Selects the noise bandwidth mode in the power sweep list table. Default Fixed.
List Points Sets and returns the number of power points to measure for the power sweep list measurement. Default 11.
List Receiver Attenuation Sets and returns the receiver attenuation for the specified row in the power sweep list table. 0dB.
List Receiver Attenuation Mode Selects the receiver attenuation mode in the power sweep list table. Default Fixed.
List Source Attenuation Sets and returns the source attenuation for the specified row in the power sweep list table. Default 0dB.
List Source Attenuation Mode Selects the source attenuation mode in the power sweep list table. Default Fixed.
Ramp Level Start Sets and returns the starting power level for the power sweep ramp measurement. Default -20dBm.
Ramp Level Stop Sets and returns the stop power level for the power sweep ramp measurement. Default -10dBm.
Ramp Nbw Auto Enable or disable increasing noise bandwidth at high powers automatically for faster measurements.
Ramp Points Sets and returns the number of power points to measure for the power sweep ramp measurement. Default 11.
Sweep No Power sweep row number. If unspecified, value is set to 1.
