Attributes

Name Description
Dither Set and read the ON or OFF state of the FFT grid dither setting
Filter Set and read the ADC filter cutoff frequency.
Filter Auto Set and read how the ADC filter is set. When ON, the ADC filter is set based on the start and stop frequencies and the ADC sampling frequency.
Mrec Period Set and read the period to wait between ADC record chunks.
Mrec Size Set and read the size of the ADC record chunks.
Mrec State Set and read the ON or OFF state of the multiple recording function.
Overload Percent Set and read the ADC overload threshold percentage. Default 80.
Record Size Force Set and read the ADC record size mode.
Record Size Force Value Set and read the integer size value for the force ADC record size feature.
Sample Decimation Fir Set and read the FIR filter for 25 MHz decimation.
Sample Rate Set and read the IQ sample rate.
Sample Rate Auto Set and read the ADC sample rate mode.
Stacking State Set and read the ON or OFF state of the ADC sample stacking.
Stacking Value Set and read the ADC stacking value.
