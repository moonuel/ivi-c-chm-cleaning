Attributes

Name Description
Frequency Sets and returns the frequency range to use for receiver leveling. Default Auto.
IFBW Sets and returns the IFBW to be used for leveling sweeps. Default 100 kHz.
IFBWEnabled Sets and returns the state of a separate IFBW setting for leveling sweeps. Default 1.
Leveling Sets all ports to pre-sweep or point leveling mode for the specified channel. Default PreSweep.
LevelingEnabled Sets and returns the state of Receiver Leveling for the specified source port. Default 0.
LSPCEnabled Sets and returns the state of Use Last Result for Source Power Cal. Default 0.
MaxIterations Sets and returns the maximum iterations to be used in order to achieve the tolerance setting. Default 10.
MaxIterationSearchEnabled Enables or disables the receiver leveling maximum iteration search function set. Default 1.
Offset Sets and returns the offset of the frequency aperture used to measure the signal power. Default 0.
Reference Sets and returns the reference receiver to be used with Receiver Leveling.
SafeModeEnabled Sets and returns the state of Safe Mode. Deafult 0.
SafeModeMaxPower Sets and returns the maximum power level for Safe Mode. Default 30 dB.
SafeModeMinPower Sets and returns the minimum power level for Safe Mode. Default -95 dB.
SafeModeStepPower Sets and returns the maximum step power level for Safe Mode. Default 1 dB.
Tolerance Sets and returns the tolerance value for leveling sweeps. Default 1 dB.
