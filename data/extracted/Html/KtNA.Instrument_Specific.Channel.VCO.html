Attributes

Name Description
Attenuation Set and read manual attenuation setting. Default 20.
AttenuationAutoEnabled Enable and disable the auto calculate attenuation setting. Default 1 (On).
FrequencyBand Set and read the frequency band. Default R8G.
FrequencyResolution Set and read the frequency resolution. Default R10.
InputLevelAutoEnabled Enable and disable the auto leveling function at pre-measurement. Default 1 (On).
MaxInputLevel Set and read the maximum input level used to calculate attenuation setting in dBm. Default 0.
PowerCorrectionEnabled Enable and disable the power correlation. Default 0 (Off).
SweepType Set and read the sweep type. Default VControl.
