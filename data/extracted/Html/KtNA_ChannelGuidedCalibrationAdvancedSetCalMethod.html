KtNA_ChannelGuidedCalibrationAdvancedSetCalMethod Function
Advanced See Also

Specifies the calibration method for each port pair.

Function Tree Node: \KtNA\Instrument Specific\Channel\GuidedCalibration\Advanced\Set Cal Method
Declaration: KtNA.h
Implementation: KtNA.dll

Syntax

Visual C++
ViStatus KtNA_ChannelGuidedCalibrationAdvancedSetCalMethod(
	ViSession Vi,
	ViConstString ChannelName,
	ViInt32 Port1,
	ViInt32 Port2,
	ViConstString CalType
);

Parameters

Vi

The ViSession handle that you obtain from the KtNA_init or KtNA_InitWithOptions function.The handle identifies a particular instrument session.

ChannelName

The physical or virtual repeated capability identifier. Pass VI_NULL or empty string if operation does not apply to a repeated capability.For valid values, see the Channel repeated capability.

Port1

First port of the pair to be calibrated.

Port2

Second port of the pair to be calibrated.

CalType

Cal type for the port pair. NOT case-sensitive. Choose from: TRL, SOLT, QSOLTN, EnhRespN or TransRespN. Replace N with the port to be used as the source port, which MUST be one of the port pair.

Return Value


Success or failure code.

Remarks

•Sending this command will overwrite the VNAs SmartCal determinations for the most accurate cal method for your connector settings and Cal Kits. Send this command ONLY if you have a deliberate reason for overwriting the SmartCal logic. You can send GuidedCalibrationAdvancedGetCalMethod to learn the cal method determined by SmartCal.
•After sending this command, send GuidedCalibrationAdvancedGetCalMethod to be sure that the command was accepted. If not, then the chosen Cal method is not compatible with the specified Thru method. For example, if the specified Thru method is Unknown Thru, an attempt to set Enhanced Response Cal should be rejected.

Instrument Command

SENSe{Channel}:CORRection:COLLect:GUIDed:PATH:CMEThod {Port1} , {Port2} , "{CalType}"
