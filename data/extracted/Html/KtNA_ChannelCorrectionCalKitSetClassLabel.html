KtNA_ChannelCorrectionCalKitSetClassLabel Function
CalKit See Also

Defines the label for the standard classes.

Function Tree Node: \KtNA\Instrument Specific\Channel\Correction\CalKit\SetClassLabel
Declaration: KtNA.h
Implementation: KtNA.dll

Syntax

Visual C++
ViStatus KtNA_ChannelCorrectionCalKitSetClassLabel(
	ViSession Vi,
	ViConstString ChannelName,
	ViInt32 CalClass,
	ViConstString Label
);

Parameters

Vi

The ViSession handle that you obtain from the KtNA_init or KtNA_InitWithOptions function.The handle identifies a particular instrument session.

ChannelName

The physical or virtual repeated capability identifier. Pass VI_NULL or empty string if operation does not apply to a repeated capability.For valid values, see the Channel repeated capability.

CalClass

Specifies the calibration class.

Label

Label of standard.

Return Value


Success or failure code.

Instrument Command

SENSe:CORRection:COLLect:CKIT:CLABel {SA|SB|SC|THRU|FWDT|FWDM|REVT|REVM|TRLT|TRLR|TRLL|UTHR|ISOL} , "{Label}"
