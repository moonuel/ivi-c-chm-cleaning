Attributes

Name Description
Active Marker Sets and reads the active marker. Default 1.
Bucket No Sets and reads the data point (bucket) number of the trace on which the marker resides.
Coupling Enabled Sets and reads the state of Coupled Markers (ON and OFF).
Coupling Method Sets and reads the scope of Coupled Markers.
Delta Enabled Specifies whether marker is relative to the Reference marker or absolute.
Display Discrete Data Enabled Makes the specified marker display either a calculated value between data points (interpolated data) or the actual data points (discrete data).
Format
Readout Position X Sets or returns the X-axis position of marker readouts. Readouts are right-justified at the specified position.
Readout Position Y Sets or returns the Y-axis position of marker readouts. Readouts are top-justified at the specified position.
Reference Mode Enabled Turns ON or OFF the reference marker mode, for the active trace of selected channel.
X Position Set or get the marker's X-axis value (frequency, power, or time).
X Reference Position Sets and returns the absolute x-axis value of the reference marker.
