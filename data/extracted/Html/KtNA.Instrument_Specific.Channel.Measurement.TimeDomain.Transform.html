Attributes

Name Description
Alignment Sets the way the PNA computes the DC value of the frequency-domain measurement.
Center Sets or returns the center time (in seconds) for time domain measurements.
Enabled Turns the time domain transform capability ON or OFF.
Kaiser Bessel Window Width Sets the parametric window width for Kaiser Bessel window in seconds.
Mode Sets the type of time domain measurement.
Parameters To Couple Specifies the time domain transform parameters to be coupled. The settings for those parameters will be copied from the selected measurement to all other measurements on the channel.
Span Sets or returns the span time (in seconds) for time domain measurements.
Start Sets or returns the start time (in seconds) for time domain measurements.
Start Stop Limits Enabled Turns the start/stop time limits ON or OFF. When ON, limits the start/stop times to avoid aliasing.
Stop Sets or returns the stop time (in seconds) for time domain measurements.
Window Impulse Width Sets the impulse width for the transform window in seconds.
Window Step Rise Time Sets the step rise time (in seconds) for the transform window.
Window Type Sets the window type for time domain transform.
