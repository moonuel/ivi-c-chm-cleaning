# Plan: Hello World Program

Goal: Add a simple "hello world" program to the project.

## Approach

Create `src/ivi_chm/hello.py` with a `main` function that prints "Hello, World!" to stdout. This keeps it within the existing `ivi_chm` package structure and follows the project's `src/` layout convention (consistent with `cli.py`, `parser.py`, etc.).

## Steps

1. Create `src/ivi_chm/hello.py`
   ```python
   def main() -> None:
       print("Hello, World!")
   ```

2. Add a corresponding test in `tests/test_hello.py`
   ```python
   from ivi_chm.hello import main

   def test_hello(capsys):
       main()
       captured = capsys.readouterr()
       assert captured.out.strip() == "Hello, World!"
   ```

3. Verify by running `pytest tests/test_hello.py`

No changes needed to `pyproject.toml` (the package already exposes `src/ivi_chm/`).
